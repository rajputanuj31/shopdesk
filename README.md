# Spur AI Live Chat Agent

An AI-powered customer support chat widget built for a fictional e-commerce store ("shopdesk"). The AI agent answers questions about shipping, returns, and store policies using OpenAI.

## Tech Stack

- **Backend**: Node.js + TypeScript + Express
- **Database**: SQLite (`better-sqlite3`, raw SQL)
- **LLM**: OpenAI `gpt-4o-mini`
- **Frontend**: React + Vite (TypeScript)
